# Incident Response Case Study

## Overview
This project presents a simulated incident response case study covering multiple cybersecurity incidents, including ransomware, phishing, and unauthorized access to sensitive data. The incidents were documented using a structured incident response approach aligned with industry best practices.

## Scope
The case study includes:
- Ransomware incident analysis
- Phishing email detection and escalation
- Unauthorized access and data exposure
- Supporting network traffic analysis

## Approach
Each incident was analyzed across key incident response phases, including detection, analysis, containment, eradication, and recovery. The focus was on clear documentation, impact assessment, and actionable mitigation recommendations.

## Key Skills Demonstrated
- Incident detection and analysis  
- Incident response documentation  
- Security risk identification  
- Mitigation and control recommendations  
- Alignment with structured incident response processes
